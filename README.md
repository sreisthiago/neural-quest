\# Neural Quest - Engine Oráculo V5



Projeto de mapeamento cognitivo para RH e instituições de ensino.



\## 🚀 Tecnologia

\- \*\*Frontend:\*\* React + Vite

\- \*\*Linguagem:\*\* TypeScript (Engine transposta do Python)

\- \*\*Estilização:\*\* Tailwind CSS



\## 🛠️ Como rodar

1\. Instale as dependências: `npm install`

2\. Inicie o projeto: `npm run dev`
